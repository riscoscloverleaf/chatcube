TBX is a C++ class library to help with the creation of RISC OS applications that use the RISC OS Toolbox modules.

This is version 0.7.5 (Alpha version). It has a few missing features and may contain some rough edges and bugs. Please send me feedback to improve this library. Contact details are in the !TBX application Help file.

TBX is released under the permissive X11 licence so can be used with commercial and non-commercial applications without charge.

TBX is provided as a Package named TBX-Dev for the the RISC OS Packaging Project.

Two additional packages have also been provided

  • TBX-Examples - simple examples showing some of the features of TBX
  • TbxAppGen - an application to generate shell TBX based applications that you can then add your own code to.

In addition for C++ and Toolbox developement you will need

  • GCC4 for compiling C++ programs from GCC for RISC OS at www.riscos.info
  • ResEdit for editing RISC OS Toolbox resources which is available as a package or the RISC OS Open website

If you do not wish to use a package manager you can download the TBX packages directly using the following links.

  • http://www.riscpkg.org/arm/TBX-Dev/TBX-Dev_0.7.5-1
  • http://www.riscpkg.org/arm/TBX-Examples/TBX-Examples_0.6-1
  • http://www.riscpkg.org/arm/TbxAppGen/TbxAppGen_0.4-1

Once downloaded drag the files to !SparkPlug or set the filetype to Archive and double click on it. You can the the drag the library/applications out of a subdirectory of the Apps folder in the archive.

Please see the Help file for the !TBX and !TbxAppGen applications or the !!ReadMe file in the TbxExamples for what to do now.

The TBX documentation file requires a browser to have been seen by the filer.
